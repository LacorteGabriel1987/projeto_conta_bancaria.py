Controle de Saques em Python
Este é um projeto simples em Python que implementa um controle de saques em uma instituição financeira. Ele monitora o número de saques realizados por um cliente e verifica se excedeu um limite predefinido.

Funcionalidades
Verifica se o número de saques excede um limite predefinido.
Auxilia os clientes no gerenciamento de suas finanças.
Código simples e fácil de entender, ótimo para iniciantes em Python.
Como Usar
Clone o repositório para o seu ambiente local.
Execute o script controle_saques.py.
Insira o número de saques realizados.
O programa indicará se o número de saques excedeu o limite.
